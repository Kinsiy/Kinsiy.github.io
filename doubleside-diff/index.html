双端 Diff 算法

发表于 2022-09-29 更新于 2023-04-09 阅读次数：

文章依据《Vue.js设计与实现》第十章 - 双端 Diff 算法 摘录而来, 更为详细的内容参见书籍原文

双端 Diff 的算法指的是, 在新旧两组子节点的四个端点之间分别进行比较, 并视图找到可复用的节点.相比简单 Diff 算法, 双端 Diff 算法的优势在于, 对于同样的更新场景, 执行的 DOM 移动操作次数更少
